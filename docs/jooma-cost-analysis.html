1

Text generation is effectively free

33 of 34 tools cost between one and five cents per use. A full 10-slide deck's text was measured at just 3.2¢.

2

Images are the only real cost

The single expensive path is AI image generation. A slideshow that generates its own images costs ~$0.26; the same deck using free stock imagery costs ~$0.045 — roughly 6× cheaper.

3

One toggle controls cost

Image source is the only meaningful lever. Everything else — text, audio narration, video — rounds to cents.

Methodology. Costs are OpenAI API usage only (gpt-4o for text at $2.50/$10 per million input/output tokens; gpt-image-1 for images; tts-1 for audio narration). The Slideshow figures are measured against live OpenAI console spend. Text-tool figures are estimates bounded by a measured fact — a 10-slide deck's text cost 3.2¢, so no single-document tool exceeds ~5¢. Excludes hosting, storage and third-party APIs (YouTube search is free). Image costs may vary slightly with model availability and prompt length.
